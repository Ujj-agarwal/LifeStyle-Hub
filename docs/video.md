## 🧰 Tech Used
- **Frontend:** React, Tailwind CSS, React Router  
- **Backend:** Flask, SQLAlchemy, Flask-JWT-Extended  
- **AI:** Google AI Studio (Gemini API)  
- **Hosting:** Frontend on Vercel, Backend on Render  
- **Auth:** JWT-based login, protected routes
